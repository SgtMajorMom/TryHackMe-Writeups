## 🎯 Purpose  
This repository documents my progress through TryHackMe’s learning paths — from foundational security concepts to advanced SOC and Red Team operations.  
It serves as both a personal study archive and a professional showcase of applied skills in threat detection, analysis, and secure system design.

---

Overview
The Secure AI Systems module provides a defender focused introduction to securing AI and machine learning systems. It explains how AI components expand the enterprise attack surface, how attackers target these systems, and how to apply structured threat modelling to identify weaknesses. The module emphasizes that AI systems require specialized security considerations beyond traditional application security.

Learning Objectives
By completing this module, I learned how to:
•	Understand AI systems as an attack surface
•	Identify weaknesses in AI/ML pipelines and integration points
•	Recognize LLM specific security threats
•	Apply STRIDE, OWASP, and ATLAS frameworks to AI threat modelling
•	Perform reconnaissance on AI systems by identifying exposed ML services and metadata
•	Conduct a structured threat modelling assessment on an AI powered application
